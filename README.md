# 📚 Indie Bookstore Landing Page

A responsive and accessible static landing page for an Indie Bookstore built using **HTML5, CSS3, and Vanilla JavaScript**. The project follows modern web development practices with responsive design, form validation, accessibility, loading indicators, and a clean monochromatic UI.

---

## 🌐 Live Demo

🔗 **Live Website:** https://indie-bookstore-landing-page.vercel.app/

---

## 🚀 Features

- Responsive Landing Page
- Modern Hero Section with Background Image
- Sticky Navigation Bar
- Bookstore Logo
- Featured Books Section
- About Section
- Contact Form
- Client-side Form Validation
- Empty State Handling
- Loading Indicator
- XSS Input Sanitization
- Analytics Console Logging Simulation
- Keyboard Accessible Navigation
- Semantic HTML5
- Mobile Friendly Design

## 🎯 Functionalities

- Displays bookstore information through a responsive landing page.
- Smooth navigation between sections.
- Highlights invalid form fields.
- Prevents empty form submissions.
- Displays loading state during form submission.
- Shows an analytics log in the browser console after user interaction.
- Sanitizes user inputs before processing.
