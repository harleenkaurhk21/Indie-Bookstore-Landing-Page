# Prompt

## Project Title
Indie Bookstore Static Landing Page

## Objective
Build a responsive and accessible static landing page for an Indie Bookstore using HTML5, CSS3, and Vanilla JavaScript.

## Requirements
- Create a clean and modern landing page.
- Include a header with logo and navigation menu.
- Design a hero section with a background image and call-to-action button.
- Display featured books using responsive cards.
- Add About and Contact sections.
- Validate form inputs and highlight invalid fields.
- Show a loading indicator during form submission.
- Display a "No data found" message for empty states.
- Sanitize user inputs to prevent XSS attacks.
- Log an analytics message in the browser console after user interactions.
- Ensure the website is responsive, accessible, and keyboard-friendly.
- Use only HTML, CSS, and Vanilla JavaScript without external libraries.

## Expected Outcome
A responsive, user-friendly, and accessible bookstore landing page that follows modern web development best practices and provides a smooth user experience across desktop and mobile devices.
